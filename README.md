# 💻 Sistema de Cadastro de Usuários

Projeto desenvolvido por Arthur Gonçalves com o objetivo de praticar lógica de programação em Python.

## 🚀 Funcionalidades
- Cadastro de usuários
- Listagem de usuários
- Exclusão de usuários
- Menu interativo no terminal

## 🧠 Tecnologias utilizadas
- Python

## 📚 Aprendizados
- Aprendi a criar um sistema com menu interativo
- Desenvolvi lógica para cadastro, listagem e exclusão de usuários
- Utilizei listas e dicionários para armazenar dados
- Pratiquei estruturas de repetição e condicionais
